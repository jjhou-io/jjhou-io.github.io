/*
我於 1996~1998 年，在元智大學資訊工程系，每學年第一學期開一門
「物件導向 Windows 程式設計（using MFC）」課程。以下是相關資料。

科目名稱（中文）：物件導向 Windows 程式設計 (使用 MFC)
科目名稱（英文）：Object Oriented Windows Programming (using MFC)
授課時間：3 小時/每週（選修）
時間：每週四 18:30pm ~ 21:20pm
地點：工程一館（資訊系館）

1. 課程目標
   Windows 是現今流行的作業系統，在個人電腦上被普遍使用。本課程教授
   物件導向 Windows 程式設計技術，並佐以 Windows 作業系統之基礎觀念，
   期使學生在作業系統與程式設計兩方面都能獲得良好的知識基礎。本課程
   選用 MFC 做為物件導向類別函式庫 (class library)。

2. 先修科目：
   C++ programming language

3. 課程大綱
   Windows 作業系統概論 (Windows 3.x and Windows 95/98)
   開發工具 (SDK v.s. MFC and OWL)
   Windows 程式運作之基本觀念
   C++ 重要論題 (封裝、繼承、動態繫結)
   認識 Visual C++ 開發工具 (機房實習)
   認識 MFC (Microsoft Foundation Class)
   一個最簡單的實例 (機房實習)
   MFC 內部運作
   Document/View 架構
   訊息映射與訊息遞送
   對話盒
   UI 物件之變化
   列印與預覽
   各種應用...

4. 上課用書
   o. 《深入淺出 MFC》, by 侯俊傑, 松崗

5. 參考書
   o. Inside Visual C++, by David Kraglinski, Microsoft Press
   o. Programming Windows95 with MFC, by Jeff Prosise, Microsoft Press

6. 教學方式
   以教本、講義、投影片、程式實例，進行教學

7. 評分：期末作業一個

*/
