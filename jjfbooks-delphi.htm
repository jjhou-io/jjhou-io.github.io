侯俊傑 序

我和達智緣起於一個偶然。

1997 年 9 月，我對於國內電腦技術寫作環境的現況，以及寫作人才的單薄，有一些感觸，在網路上發表了一篇名為「拋磚引玉」的文章。發表之後很得了一些朋友的來信，其中有達智。

當時他正在撰寫一本 Delphi 書籍，定位在比較深層紮實的技術面上。由於 Delphi 是一套「軟體快速開發工具」（RAD，Rapid Application Development），拖拉點放之間幾乎可以輕鬆完成一些還不錯的程式，因此許多學習者沒有養成良好的習慣，沒有奠定紮實的基礎，卻將自己放逐（或沉醉）於浮光掠影之間，殊為可惜！這種現象造成愈來愈多的人對於 RAD 工具及其使用者的誤解。同時，由於心力付出和現實收穫之間的巨大不平衡，也使得市面上有關 Delphi 的深度好書寥寥可數。

於是，一本用來為 Delphi 使用者奠定基礎的書籍，更顯得彌足珍貴。

達智的構想是要在 Delphi 領域撰寫一本類似 Programming Windows 95 那種全盤定位、全面掌握的厚實大書，詳細闡述 VCL 的工作原理，及其與 SDK programming 的關係。因為他的用心、他的實力、以及他的構想，我很樂意給他意見，並同意擔任最後的校潤總監工作。其後，由於內容的重新組織，構想中的書籍分裂為三本，分別定位在基礎、進階、以及資料庫三方面。

定位於基礎面的本書，全稿完成之時，我剛好因為個人因素，沒有時間擔任最後的總校潤工作，深感遺憾。然而基於我對達智的瞭解，以及他個人的實力和用心，我相信這本書一定有著我所預期的水準。

我希望，達智的另兩本書能夠如期完成，帶給更多欲進入 Delphi 領域的人以更好的知識糧食。

侯俊傑 1998/04/10 于新竹
