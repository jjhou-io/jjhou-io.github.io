《深入淺出 MFC》2/e 的重點，一在以第三章的 frame1~frame8 模擬
MFC 的架構與運行，二在利用 VC++ 所附之 Scribble 範例程式解說
MFC applications 的來龍去脈。至於其他小程式，細瑣，無足掛齒。

如果您購買《深入淺出 MFC》2/e 書籍，書後附有光碟，內含所有程式源碼。如果您下載《深入淺出 MFC》2/e 電子書，可以 PDF reader 選取檔案中的程式源碼文字，稍加整理，即可整理出可用之源碼。

為方便讀者，本處提供第三章之 frame1~frame8 程式（缺 frame5，書中有說明）。至於 chap7~chap12 所採用之 Scribble 程式可自 VC++ 光碟片中獲得。

每個 frame_ 程式皆放在獨立的 folder 中，所有 folders 再壓縮為 frame.zip 檔。每一個 folder 都有直接可用之 .cpp 和 .h 檔。每一個 frame_ 程式都可在 VC6 和 BCB4+ 順利編譯（少部份程式在 BCB4+ 有警告訊息），編譯方式如下：

☉vc6        cl my.cpp mfc.cpp <Enter>
☉bcb4+    bcc32 my.cpp mfc.cpp <Enter>

這是 command line 編譯模式，請參考【在 console 模式中使用 C/C++ 編譯器】一文。
