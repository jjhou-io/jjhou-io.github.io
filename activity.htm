Module1 C++底層機制剖析與實踐 (2 days, 12 hours)
課程綱要：內容涵蓋 source code --(compile)--> object code --(link)--> executable code --(load)--> process 的完整內部過程與設計，深入解析 object file format, executable file format, process' memory layout, static link, dynamic link, C++ startup code, C++ dynamic binding, underlying implementation of C++. 其中許多觀念不獨面向 C++，也面向其他編程語言。

Module2 C++ Memory 管理及優化 (2 days, 12 hours)
課程綱要：內容涵蓋 (1) malloc/free 詳細運作與源碼剖析，讓我們從中了解已經做了什麼，還可能需要做些什麼，(2) 立於 malloc/free 之上的種種內存管理作法，包括 C++ 標準庫、MFC、Boost、Loki 的經典作法。(3) 內存管理相關的所有 C++ primitives.

Module3 深度剖析 STL & GP 的設計與應用 (2 days, 12 hours)
課程綱要：深度剖析 GP（泛型編程）最重要最成功的產品：STL / C++ Standard Library，並以此為基礎，尋求高階應用上正確選擇 C++ 標準庫的容器（containers）、算法（algorithms），乃至自行開發適配器（adapters）、迭代器（iterators）。

課程面向高階程式員，目標：1. 對於技術，知其然並知其所以然     2. 將底層的、內核相關的知識應用於項目改善，或提昇個人及團隊的技術水平。


■ C++ STL 與泛型編程之設計及實踐 高級培訓

培訓簡介：徹底理解 C++ 標準庫（以下簡稱 STL）之設計與實現，是良好運用 STL 的先決條件與較高起點，而良好運用 STL又是現今所有 C++ 編程不可或缺的一環，也是事半功倍的最關鍵環節。太多軟件由於選擇了不當的容器或不當的算法，造成性能低下。本課程包括 C++ Templates 全面鍛鍊，STL 之總體設計、數據結構與實現技術，從而讓學員具備正確選擇容器和算法，乃至擴充 STL 的能力。
培訓大綱 (2 天, 12 小時)

Part 1：C++ Template 技術
* Generic Programming (泛型編程) 概觀
* C++ Template (模板) 概觀
* Class Template (類模板) and Function Template (函數模板)
* Operator Overloading (操作符重載)

Part 2：STL 之總體思想、結構設計、關鍵源代碼剖析
* Allocator 分配器
* Container 容器
* Algorithm 算法
* Functor 仿函式
* Adapter 適配器
* Iterator 迭代器

Part 3：STL 之檢討、選擇、擴充、改良

■ C++ 底層機制剖析

培訓簡介：本課程涵蓋 general C++ programming 之外的多重議題，主旨在於讓 C++ Programmer 對於 C++ 有更深入的了解和體會。探討的主題不一定能立刻反應於實際編程之中 (取決於編程領域)，但對程序員的技能涵養的提昇很有幫助。

培訓大綱 (2 天, 12 小時)
Part 1：C++ 編譯期和運行期綜論
* 涵蓋 Compile (編譯) => Object File (目的文件) => Link (鏈接) => Executable File (可執行文件) => Load (裝載) => Process (進程) 的內存佈局

Part 2：Startup code (啟動碼) 詳論，兼談 CRT (runtime library)
* 深度剖析 startup code 的作用、改編方式、代碼細節 (以 VC's mainCRTStartup 為分析對象)
* 涵蓋 CRT 對 memory leak 的監控機制 (Heap State Reporting Functions)
* 涵蓋 debug mode 對 C++ Object 的掌控機制 (Debug Heap)
* 探討 Objects Dumping 作法 (以 MFC 的 CDumpContext, CMemoryState, CObject::Dump 為例)

Part 3：C++ Under the Hood (C++ 幕後機制)
* From C to C++：以 C 模擬 C++ class, class members, 'this' pointer，揭示 Object Oriented (面向對象) 的幕後機制。
* Object Model (對象模型) 剖析
* Virtual Mechanism (虛機制) 剖析
* Polymorphism (多態) and Dynamic Binding (動態綁定) 剖析
* Stack/Heap Overflow —— 淺談黑客手法
* Others Underlying implementation in C++


■ C++ 內存管理 (Memory Management)

培訓簡介：C++ 允許 (並且要求) 程序員自行操控 Memory Heap (i.e. Dynamic Allocation)，這雖然帶給程序員負擔，從某個角度說卻也是 C++ 的優勢 (速度表現與空間自主)。為此，我們必須追求在此現實下如何使用 memory 以符合空間和速度上的最佳利益。本課程在低階面全面介紹與 memory 相關之 C++ 語言構件，在高階面深度剖析各大程序庫 (STL, Loki, MFC, Boost) 的 pooled allocation 作法，並檢討它們的優缺點，作為可能的借鑑或改善方向。

培訓大綱 (2 天, 12 小時)

Part 1：malloc/free：這是 C/C++ 動態分配的根基和起點。此部分讓學員充份了解其中的精巧繁複，使足以評估在此之上添加內存管理的必要性與可能方向。
* 圖解 malloc/free 的相關結構，及內存分配過程中的變化。
* 剖析 malloc/free 的關鍵源代碼
* 以 Visual C++ 和 GCC 為剖析對象

Part 2：C++ 內存管理相關構件和重載方式
* new expression vs. operator new
* delete expression vs. operator delete
* array new and array delete
* placement new
* new/delete operator overloading

Part 3：實例分析 (含關鍵源碼分析)
* Allocator in C++ Standard Library
* Allocator in Loki
* Allocator in MFC
* Allocator in Boost
