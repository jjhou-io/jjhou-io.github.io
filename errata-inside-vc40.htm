深入 Visual C++ 4.0   勘誤

原著：Inside Visual C++ 4.0, by David Kruglinski, Microsoft Press/1996

過去，我對每本作品（著譯皆然），總是把握新刷機會，修正書中的錯誤或不妥。偶也在不影響頁次的情況下，做一些內容補充。但是並未有系統地將這些動作記錄下來。

由於經驗的累積，逐漸知道，應該有系統地將書籍的各刷變化提供給讀者。

在我陸續建立起晚近各書的 errata 資料後，有不少讀者關心 <深入 Visual C++ 4.0> 的 errata。於是我回頭整理下來，供讀者參考。

由於我手上沒有保留 <深入 Visual C++ 4.0> 各刷版本，無從將各刷變化記錄下來，僅就印象所及的修正，列出於後。或許您手上的書籍已經沒有這些錯誤。

以下 Ln 代表第 n 行，L-m 代表倒數第 m 行。

頁次：70, L6
原文：如果上述例子中你以 MM_ANISOTROPIC 取代 MM_ISOTROPIC，橢圓形...
更正：如果上述例子中你以 MM_ISOTROPIC 取代 MM_ANISOTROPIC，橢圓形...

頁次：71, L-12
原文：// m_rect 是 CRect 的資料成員，CMyView 設定 MM_LOENGLISH 映像模式
更正：// m_rect 是 CMyView 中的資料成員（型別為 CRect），用以記錄 MM_LOENGLISH 座標

頁次：86, 中段程式碼 L1
原文：void CMyView::OnDraw(DC* pDC)
更正：void CMyView::OnDraw(CDC* pDC)

頁次：87, 上段程式碼 L1
原文：void CMyView::OnDraw(DC* pDC)
更正：void CMyView::OnDraw(CDC* pDC)

頁次：87, 上段程式碼 L4
原文：CPen* pOldPen = pDC->SelectObject(&newPen);
更正：pDC->SelectObject(&newPen);
注意：其實原來的那行也不會造成程式錯誤。

頁次：p178, L-9
原文：讓我做過假設：你發現了...
更正：讓我做個假設：你發現了...

頁次：p179, L6
原文：事實上，當 Developer Studio 呼叫一個特殊的 expoerted 函式，...
更正：事實上，當 Developer Studio 呼叫一個特殊的 exported 函式，...

頁次：p185, L6
原文：...當你在圖形編輯器
更正：...當你在資源編輯器

頁次：p193
注意：我在本頁最上有個譯註，說螢幕上看不到【Category】combobox，所以...
更正：按下 p192 圖右側第二個按鈕【Add Variable...】，便可見【Category】combobox）

頁次：chap10, class CDib source in CD-ROM
注意：原文書所附的 class CDib（原始碼未列於書中）有嚴重錯誤。
更正：請參考文章 Serialize 的聯想 (1)
