■2005.02.05 聯合報 (A12)：
「愛因斯坦能獲得人類科學發展中最偉大的成就，在於他花了許多時間發掘問題」。中央研究院院長李遠哲昨天表示，臺灣學校教育把課程排得滿檔，學生沒時間去想問題，且大學教授薪資待遇太低，留不住人才，都值得深思檢討。


■Sent: Monday, November 22, 2004 12:15 AM
...公司對我的辭職傷透腦筋，因為，為了尋找能接替我工作的新人，公司找了一個禮拜，面試了廿位號稱懂 C++ 的工程師，結果沒有一個是真正懂的。許多 C++ 的基本觀念，面試者都答不出來，令我對台灣程序員的軟體水準非常擔憂。我找了一位朋友，希望他能幫我找 C++ 程序員，他說 :「104 不是有一堆會 C++ 的工程師，怎麼不去 104 找看看？」我告訴他說已經找了廿位號稱會 C++ 的工程師，還是找不到合適的人，他非常驚訝地說 :「我以為台灣會 C++ 的人很多。」我說：「我本來也是這麼認為啊！」

我覺得很好奇的是，連 class 都不太會用，多型與虛擬的觀念也不甚了解的人， 怎麼好意思說自己懂 C++ 呢？當人力市場充斥這種人時，台灣的軟體水準如何能獲得發展呢？真是令人擔憂啊！
